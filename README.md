# covid19knowledgebase
It is the knowledgebase for COVID-19 related questions with links. **Not prodiction ready**

## Run
#### Prerequirements
- python version 3.8.1
- pip version 20.0.2 
- flask version 1.1.1

#### Once
##### **Windows, x64**
- `set FLASK_APP=flaskr`
- `set FLASK_ENV=development`

#### Start
- `flask run`
